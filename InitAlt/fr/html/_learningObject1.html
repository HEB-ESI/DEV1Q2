1.1.1.1. Compréhension d’algorithme

Structure d'un programme

Le morceau d'algorithme suivant est-il correct ? 
lire n
// n est un entier
selon que n vaut
1, 2, 3, 4 : n ← 3*n
3, 5, 7, 9 : n ← 2*n
fin selon
?
?
Vous n'avez pas encore marqué toutes les réponses correctes!

Pour ces exercices, nous vous demandons de comprendre des algorithmes donnés.

Compréhension

Que vont-ils afficher ? 
  • module exerciceA()
    x : entier
    ok : booléen
    lire x
    ok ← x > 2
    si ok alors
      afficher 2*x
    sinon
      afficher 3*x
    fin si
fin module

    Si le nombre lu est 2 ?

    ,6,6
  • module exerciceB()
    a,b,c : entier
    lire b,a
    si a > b alors
      c ← a DIV b
    sinon
      c ← b MOD a
    fin si
    afficher c
fin module

    Si les nombres lus sont respectivement 2 et 3 ?

    ,1,1

    Si les nombres lus sont respectivement 4 et 1 ?

    ,0,0

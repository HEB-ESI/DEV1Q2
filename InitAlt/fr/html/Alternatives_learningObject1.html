Complétez la condition manquante pour donner du sens au code suivant : 

algorithme multiple5(n : entier) → chaine
    aRetourner : chaine
    si  ,n MOD 5 = 0,n MOD 5 = 0,n MOD 5=0 alors
        aRetourner ← n, "est divisible par 5"
    sinon
        aRetourner ← n, "n'est pas divisible par 5"
    fin si
    retourner aRetourner
fin algorithme

1.2.4. Appel de méthode

Une méthode est une boite noire :

Pour l’utiliser, il faut connaitre :

  • son nom ;
  • ce dont elle a besoin ;
  • ce qu’elle retourne ;
  • mais pas comment elle fait ;

On peut appeler une méthode

  • à partir du code d’une autre classe : NomClasse.nomMéthode(...)
  • au sein de la même classe : nomMéthode(...)
double racine = Math.sqrt(4.0); double aléatoire = Math.random(); int nb = −10; int absolu = Math.abs(nb);

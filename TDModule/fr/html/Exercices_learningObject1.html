1.7.1. Compréhension d’algorithme

Pour ces exercices, nous vous demandons de comprendre des algorithmes donnés.

Compréhension

Que vont-ils afficher ? 
  • algorithme ex1 ()
    x, y : entiers
    addition(3, 4, x)
    afficher x
    x ← 3
    y ← 5
    addition(x, y, y)
    afficher y
fin algorithme

algorithme addition(a↓, b↓, c↑ : entiers)
    somme : entier
    somme ← a + b
    c ← somme
fin algorithme
    ,7,7 ,8,8

  • algorithme ex3 ()
    a, b, c : entiers
    calcul(3, 4, c)
    afficher c

algorithme calcul(a↓, b↓, c↑ : entiers)

    b ← 3*b
    c ← a+b
fin algorithme

  • algorithme ex4 ()
    a, b, c : entiers
    a ← 3
    b ← 4
    c ← f(b)
    afficher c
    calcul2(a, b, c)
    afficher a, b, c
fin algorithme

algorithme calcul2 (a↓, b↓, c↑ : entiers)
    a ← f(a) 
    c ← 3*b 
    c ← a+c
fin algorithme

algorithme f (a↓ : entier) → entier
    b : entier
    b ← 2*a+1
    retourner b
fin algorithme
    ,9,9 ,3,3 ,4,4 ,19,19

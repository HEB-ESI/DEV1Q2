    1. TD Variables structurées

    Ces exercices ont pour but de vérifier que vous avez fixé les variables structurées.

    Rectangle

    Définir un type Rectangle pouvant décrire de façon commode un rectangle dans un espace à deux dimensions et dont les côtés sont parallèles aux axes des coordonnées. Écrire ensuite

    • un module calculant le périmètre d’un rectangle reçu en paramètre ;
    • un module calculant la surface d’un rectangle reçu en paramètre ;
    • un module recevant en paramètre un rectangle R et les coordonnées d’un point P, et renvoyant vrai si et seulement si le point P est à l’intérieur du rectangle R ;
    • un module recevant en paramètre un rectangle R et les coordonnées d’un point P, et renvoyant vrai si et seulement si le point P est sur le bord du rectangle R ;
    • un module recevant en paramètre deux rectangles et renvoyant la valeur booléenne vrai si et seulement si ces deux rectangles ont une intersection.

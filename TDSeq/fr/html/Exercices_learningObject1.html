1.8.1. Compréhension d’algorithme

Pour ces exercices, nous vous demandons de comprendre des algorithmes donnés.

Compréhension

Que vont-ils afficher si à chaque fois les deux nombres lus au départ sont successivement 2 et 3 ? 
  • module exerciceA()
  lire a, b
  b ← b+2*a
  afficher b
fin module
    ,7,7
  • module exerciceB()
  a,b : entiers
  lire a, b
  a ← a+2*b
  afficher a
fin module
    ,8,8
  • module exerciceC()
  a,b : entiers
  lire b, a
  b ← b+2*a
  afficher b
fin module
    ,8,8
  • module exerciceD()
  a,b : entiers
  quotient : réel
  lire b, a
  quotient ← a / b
  afficher quotient
fin module
    ,1.5,1.5
  • module exerciceE()
  a, b, c, d : entiers
  lire c, d
  a ← 2*c+5*d
  b ← 2+c*3+d
  c ← a MOD b
  afficher a DIV c
fin module
    ,2,2
  • module exerciceF()
  x, y : réels
  lire x, y
  x ← x*x 
  x ← x*x+y*y 
  x ← √x
  afficher x
fin module
    ,5,5
  • module exerciceG()
  x, y : réels
  lire x, x
  x ← x MOD x + (x + 1) DIV 2
  afficher x + 3
fin module
    ,5,5

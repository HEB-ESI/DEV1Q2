1.4.2. Affectation interne en Java

On parle d’affectation interne lorsque la valeur d’une variable est « calculée » par l’exécutant de l’algorithme lui-même à partir de données qu’il connait déjà : nomVariable = expression; Rappelons qu’une expression est une combinaison de variables et d’opérateurs. L’expression a une valeur.

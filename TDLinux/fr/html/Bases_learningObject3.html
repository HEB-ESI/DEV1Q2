1.1.3. Quelques commandes courantes

Les commandes de base

La commande pour :
  • voir le contenu d'un dossier (la liste de ce qu'il contient) est ,ls,ls
  • voir le contenu d'un dossier (la liste de ce qu'il contient) au format long est ,ls -l,ls -l
  • voir le contenu d'un dossier (la liste de ce qu'il contient), y compris les fichiers cachés est ,ls -a,ls -a
  • éditer le contenu d'un fichier est ,nano,nano
  • changer son mot de passe est ,passwd,passwd
  • se déconnecter de linux1 est ,exit,exit
  • copier un fichier est ,cp,cp
  • renommer un fichier est ,mv,mv
  • déplacer un fichier est ,mv,mv
  • changer de dossier courant est ,cd,cd
  • créer un répertoire est ,mkdir,mkdir
  • visualiser le contenu d'un fichier sans l'éditer est ,cat,cat
  • voir quel est le dossier courant (son chemin) est ,pwd,pwd
  • détruire un fichier est ,rm,rm
  • détruire un dossier vide est ,rmdir,rmdir,rm -r
  • détruire un dossier pas forcément vide est ,rm -r,rm -r
  • d'obtenir la liste des options de la commande rm est ,man rm,man rm

La ligne de commande

  • Qu'est-ce qui permet de distinguer / séparer les différentes parties d'une commande ? ,l'espace,l'espace,espace
  • Dans la commande nanotest, combien y a-t-il de parties ? ,1,1,un,une
  • Dans la commande nano test, combien y a-t-il de parties ? ,2,2,deux

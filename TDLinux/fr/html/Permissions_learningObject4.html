1.3.4. Rappel des tds précédents

Les commandes linux

Quelle commande linux permet de faire l'action suivante ?
  • se déconnecter ,exit
  • changer son mot de passe ,passwd
  • nettoyer l'écran ,clear
  • voir le contenu d'un dossier au format long ,ls ,-l
  • créer un répertoire ,mkdir
  • déplacer un fichier ,mv
  • connaitre les groupes d'un utilisateur ,groups
  • modifier le groupe d'un fichier ,chgrp
  • modifier le propriétaire d'un fichier (par le super-utilisateur!) ,chown
  • retirer la permission de traverser le répertoire tds à tous ,chmod ,a-x ,tds
  • placer le fichier Hello.java dans le groupe etudiants1 ,chgrp ,etudiants1 ,Hello.java

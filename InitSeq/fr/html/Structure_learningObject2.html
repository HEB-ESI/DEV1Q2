Nom d'un programme

Quels noms doivent avoir les fichiers dans lesquels sont placés les programmes suivants : 
    public class Exercice {  
        // Methods    
    }
,Exercice.java,Exercice.java
    public class SommeChiffres {
        // Methods    
    }
,SommeChiffres.java,SommeChiffres.java
    public class sommechiffres {    
        // Methods    
    }
,sommechiffres.java,sommechiffres.java
Le nom du fichier est le même nom que la classe avec l'extension ".java" Attention, pour le dernier exemple, nous avons volontairement écrit le nom de la classe sans majuscule... par convention, ce n'est pas correct!!!

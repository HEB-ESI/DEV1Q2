1.1.1. Les variables et les types

Le type des données

Comment déclarer en algorithmique :
  • un montant d'un ticket de caisse ? ,montant : réel
  • une cote sur le bulletin (pas de demi-point autorisé) ? ,cote : entier
  • le titre d'un film ? ,titre : chaine
  • l'initiale de votre nom ? ,nom : chaine

Le type des données

Comment déclarer en Java :
  • un montant d'un ticket de caisse ? ,double montant;
  • une cote sur le bulletin (pas de demi-point autorisé) ? ,int cote;
  • le titre d'un film ? ,String titre;
  • l'initiale de votre nom ? ,char nom;

Types Java

Donnez l'équivalent Java des types vus en algorithmique.

entier ,int,int
réel ,double,double
chaine ,String,String
chaine d'un seul caractère ,char,char
booléen ,boolean,boolean
